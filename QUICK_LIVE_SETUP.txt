📋 STEP-BY-STEP CHECKLIST

□ STEP 1: Get Live Keys from Razorpay
  └─ Login: https://dashboard.razorpay.com
  └─ Switch to "Live Mode"
  └─ Settings → API Keys → Generate Live Keys
  └─ Copy: Key ID (rzp_live_...) and Key Secret

□ STEP 2: Update Backend .env
  └─ Open: x:\desgin\tropical\backend\.env
  └─ Add these lines:
     
     RAZORPAY_KEY_ID=rzp_live_YOUR_KEY_ID_HERE
     RAZORPAY_KEY_SECRET=YOUR_KEY_SECRET_HERE
     RAZORPAY_WEBHOOK_SECRET=YOUR_WEBHOOK_SECRET_HERE
  
  └─ Save file

□ STEP 3: Update Frontend .env
  └─ Open: x:\desgin\tropical\frontend\.env
  └─ Add this line:
     
     VITE_RAZORPAY_KEY_ID=rzp_live_YOUR_KEY_ID_HERE
  
  └─ Save file

□ STEP 4: Restart Backend
  └─ Stop current backend (Ctrl+C)
  └─ Run: cd x:\desgin\tropical\backend
  └─ Run: npm run dev
  └─ Check for: "✓ Razorpay initialized successfully"

□ STEP 5: Restart Frontend
  └─ Stop current frontend (Ctrl+C)
  └─ Run: cd x:\desgin\tropical\frontend
  └─ Run: npm run dev

□ STEP 6: Test with Small Amount
  └─ Add product to cart
  └─ Checkout with real details
  └─ Pay ₹10 (test amount)
  └─ Verify order created
  └─ Check email received

□ STEP 7: Setup Webhook (Recommended)
  └─ Razorpay Dashboard → Settings → Webhooks
  └─ Create webhook: https://yourdomain.com/api/payment/webhook
  └─ Copy webhook secret
  └─ Add to backend .env as RAZORPAY_WEBHOOK_SECRET
